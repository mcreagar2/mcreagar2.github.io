Portfolio Projects

  • SAAM: Music Listening History
    Shiny App
    -The data that powers this app came from four separate Google sheets (Songs, Artists, Albums, Miscellaneous). Google Sheets was starting to have stability trouble since the sheets were so large (data on thousands of songs recorded each month for about 10 years). The main goal of the application is to present the data to a viewer in a clean, easy-to-access manner. The user should ideally be able to think of a song, artist, or album and see if the song is in the data set and any statistics on that song.
